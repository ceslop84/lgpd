# O que é a LGPD? 

A **LGPD** (Lei Geral de Proteção de Dados) é a lei brasileira que protege **seus dados pessoais**.

Ela entrou em vigor em **14 de agosto de 2020** e estabelece regras sobre como as empresas e organizações podem coletar, usar e guardar informações sobre você.

## 📚 Entendendo os conceitos

### O que é um "dado pessoal"?

Qualquer informação que possa identificar você. Por exemplo:
- 📝 Nome, CPF, RG
- 📞 Telefone, email
- 📍 Endereço
- 💳 Dados de cartão de crédito
- 🎂 Data de nascimento
- 👤 Fotos, vídeos
- 🌐 Histórico de navegação na internet
- 🛒 Histórico de compras

### O que é um "dado sensível"?

Alguns dados precisam de proteção ESPECIAL porque revelam coisas muito pessoais:
- 🏥 Dados de saúde e genéticos
- 🗣️ Origem ou convicção política
- ⛪ Convicção religiosa ou filosófica
- 👥 Dados sobre vida sexual ou orientação sexual
- 🏷️ Dados sobre raça ou etnia

**Dados sensíveis exigem CONSENTIMENTO EXPLÍCITO. Você deve autorizar CLARAMENTE.**

## 🏢 Quem é responsável?

Existem três personagens principais na LGPD:

### 1. **Titular dos Dados**
- É VOCÊ! A pessoa dona das informações

### 2. **Controlador**
- Quem decide por que e como seus dados serão usados
- Exemplo: A empresa Facebook/Meta

### 3. **Operador**
- Quem processa os dados SOB INSTRUÇÕES do controlador
- Exemplo: Uma empresa de análise de dados contratada

## ⚖️ Os Pilares da LGPD

A lei se baseia em 10 princípios fundamentais:

1. **Finalidade**: Dados devem ser coletados para propósitos específicos e legítimos
2. **Adequação**: As informações devem ser relevantes para o objetivo
3. **Necessidade**: Só coletar dados que realmente são necessários
4. **Livre acesso**: Você pode acessar seus dados facilmente
5. **Qualidade**: Os dados devem estar corretos e atualizados
6. **Transparência**: Empresas devem ser claras sobre o que fazem
7. **Segurança**: Empresa deve proteger seus dados contra vazamentos
8. **Prevenção**: Evitar problemas antes que ocorram
9. **Não discriminação**: Não usar seus dados para discriminar você
10. **Responsabilidade**: Empresa é responsável se algo der errado

## 📋 Bases Legais para coleta de dados

Uma empresa só pode usar seus dados se tiver uma "base legal". São várias:

```
✅ CONSENTIMENTO: Você autoriza explicitamente
✅ CONTRATO: Necessário para executar um contrato com você
✅ OBRIGAÇÃO LEGAL: Exigido por lei (ex: imposto)
✅ ADMINISTRAÇÃO PÚBLICA: Exercer políticas públicas
✅ INTERESSE LEGÍTIMO: Interesse legítimo da empresa
✅ PROTEÇÃO FÍSICA/SAÚDE: Proteger vida e saúde
✅ PESQUISA: Pesquisa científica
✅ DADOS PÚBLICOS: Informações já públicas
```
